---

## 🚀 핵심 가치 (Core Values)

- **신선함** : 당일 생산, 냉장 배송 시스템  
- **안전함** : HACCP 인증 및 엄격한 품질 관리  
- **편리함** : 맞춤형 정기 구독과 자동 결제  
- **스마트함** : AI 기반 아기 성장 단계별 식단 추천
